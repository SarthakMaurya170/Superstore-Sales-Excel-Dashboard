#  Superstore Sales Performance Dashboard

## Project Overview
This project transforms a raw, 10,000+ row dataset into an interactive Excel dashboard to track KPIs, analyze regional performance, and identify high-value product categories.

## 🛠️ Tools & Techniques Used
* **Microsoft Excel**
* **Data Cleaning:** Power Query (Resolved locale-based data type formatting)
* **Data Processing:** Time-series extraction and custom business logic (Shipping Days)<img width="1041" height="772" alt="Superstore_Sales_Dashboard" src="https://github.com/user-attachments/assets/99142216-4b03-4744-b153-19f50d5bf774" />

* **Data Modeling:** Multi-layered PivotTables 
* **Data Visualization:** Interconnected Slicers, dynamic charts, and custom KPI scorecards

##  Key Insights 
1. **Top Category:** Technology drives the highest revenue across all regions.
2. **Top Sub-Category:** Chairs and Phones consistently outperform other sub-categories. 
3. **Geographic Trend:** The West region generates the highest overall sales volume.

##  How to Use This Dashboard
1. Download the `.xlsx` file from this repository.
2. Open the file in Microsoft Excel.
3. Use the **Order Year** and **Segment** Slicers at the top to interact with the data and watch the charts and KPI scorecards update dynamically.
